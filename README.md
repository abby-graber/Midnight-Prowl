Overview:

Game Name: Midnight Prowl

Members: Abby Graber, Sneaha Santra, Siya Desale, Ananya Patchigolla

Game Details:

Genre: Puzzle, Platformer Platform: Computer / PC Players: Single Player



Installation and Setup (via ZIP):

Download and extract the zip file

Navigate to the GameMaker project in the Midnight-Prowl directory

Open and run game in GameMaker

Click on "Tutorial"

Use the arrow keys to move player

Pick up the note in the level for the rest of the instructions



Cheat Codes:

Alt + N: Takes you to the next room (not while obj_story is active)

Alt + B: Goes to the previous room

Alt + R: Resets the game

Alt + F: Toggles window to fullscreen
